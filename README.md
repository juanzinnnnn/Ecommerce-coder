# Ecommerce React Native + Expo

Proyecto final Coderhouse: aplicación de ecommerce desarrollada con React Native + Expo.

## Tecnologías principales

- React Native con Expo
- React Navigation (Stack + Tabs)
- Redux Toolkit
- Firebase Authentication
- Firebase Realtime Database
- SQLite (expo-sqlite)

## Estructura del proyecto

- src/
  - components/
  - screens/
  - navigation/
  - store/
  - services/
  - database/
  - hooks/
  - constants/
  - assets/

## Funcionalidades

- Registro e inicio de sesión de usuarios (Firebase Auth)
- Listado de categorías y productos
- Productos por categoría
- Detalle de producto
- Carrito con Redux Toolkit + persistencia local
- Perfil de usuario con foto usando cámara del dispositivo
- Persistencia offline con SQLite

## Scripts

- `npm start` / `expo start`

Configura tu `firebaseConfig.js` con los datos de tu proyecto de Firebase antes de ejecutar la app.
